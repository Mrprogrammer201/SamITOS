**Документация к коду SamITOS**

### Описание проекта
SamITOS - это простой симулятор операционной системы, реализованная в виде консольного эмулятора на Node.js. Она поддерживает базовые команды управления процессами и файлами, а также вывод информации о системе.

### Основные компоненты

1. **Класс SamITOS**
   - Содержит словарь команд `this.commands`, которые соответствуют возможным действиям в системе.
   - Поддерживает управление процессами и файлами в виртуальной среде.
   - Реализует методы для обработки команд пользователя.

2. **Методы класса SamITOS**

   - `run_command(command)`:
     - Обрабатывает введенную пользователем команду.
     - Разделяет команду на аргументы и вызывает соответствующую функцию.

   - `poweroff()`: Завершает выполнение программы (эмуляция выключения).

   - `reboot()`: Перезагружает систему, очищая файлы и процессы.

   - `echo(...message)`: Выводит переданный текст в консоль.

   - `ls()`: Выводит список файлов в виртуальной файловой системе.

   - `cr_fl(filename)`: Создает новый файл.

   - `dl_fl(filename)`: Удаляет файл.

   - `read_file(filename)`: Выводит содержимое файла.

   - `write_file(filename, ...text)`: Записывает текст в файл.

   - `cr_proc(processName)`: Создает процесс с уникальным ID.

   - `dl_proc(processId)`: Удаляет процесс по его ID.

   - `nicopat()`: Выводит информацию о системе (аналог neofetch).

   - `ps()`: Выводит список активных процессов.

   - `clear()`: Очищает консоль.

   - `help()`: Выводит список доступных команд и их описание.

3. **Функция `getCommandDescription(cmd)`**
   - Возвращает описание команд для справочной системы.

4. **Обработка ввода пользователя**
   - При запуске кода создается интерфейс `readline` для работы с консолью.
   - Выводится приветственное сообщение `---SamITOS BETA VERSION---`.
   - Программа ожидает ввода команды, выполняет ее и ждет следующего ввода.

### Принцип работы
- Пользователь вводит команду в консоль.
- Программа анализирует команду и вызывает соответствующую функцию.
- Выполняется запрашиваемое действие (например, создание файла, вывод списка процессов и т.д.).
- После выполнения команда ждет следующего ввода.
- Если вводится `poweroff` или `exit`, программа завершает работу.

### Возможности SamITOS
- Управление файлами (создание, удаление, чтение, запись).
- Управление процессами (создание, удаление, просмотр активных процессов).
- Вывод информации о системе.
- Очистка консоли и базовые утилиты.

SamITOS является простым эмулятором ОС, который можно расширять, добавляя новые команды и функциональность.
