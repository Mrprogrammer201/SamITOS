**Документация к коду SamITOS**

### Описание проекта
SamITOS - это простой симулятор операционной системы, реализованный на языке C. Он поддерживает базовые команды управления процессами и файлами, а также вывод информации о системе.

### Основные компоненты

1. **Основные структуры данных**
   - `Process` - структура для хранения информации о процессах (имя, ID).
   - `File` - структура для управления виртуальной файловой системой (имя, содержимое).

2. **Основные функции**

   - `print_samitos_ascii()`: Выводит ASCII-логотип SamITOS.

   - `print_system_info()`: Выводит информацию о системе (аналог neofetch).

   - `poweroff()`: Завершает выполнение программы (эмуляция выключения).

   - `reboot(char *argv0)`: Перезапускает систему, вызывая сам процесс заново.

   - `echo(char *message)`: Выводит переданный текст в консоль.

   - `ls()`: Выводит список файлов в виртуальной файловой системе.

   - `cr_fl(char *filename)`: Создает новый файл.

   - `dl_fl(char *filename)`: Удаляет файл.

   - `read_file(char *filename)`: Выводит содержимое файла.

   - `write_file(char *filename, char *text)`: Записывает текст в файл.

   - `cr_proc(char *processName)`: Создает процесс с уникальным ID.

   - `dl_proc(int processId)`: Удаляет процесс по его ID.

   - `ps()`: Выводит список активных процессов.

   - `clear_screen()`: Очищает консоль.

   - `help()`: Выводит список доступных команд и их описание.

3. **Обработка ввода пользователя**
   - В `main()` реализован цикл обработки команд.
   - Ввод читается с помощью `fgets()` и передается в `run_command()`.
   - `run_command()` анализирует ввод и вызывает соответствующую функцию.

### Принцип работы
- Пользователь вводит команду в консоль.
- Программа анализирует команду и вызывает соответствующую функцию.
- Выполняется запрашиваемое действие (например, создание файла, вывод списка процессов и т.д.).
- После выполнения команда ждет следующего ввода.
- Если вводится `poweroff`, программа завершает работу.

### Возможности SamITOS
- Управление файлами (создание, удаление, чтение, запись).
- Управление процессами (создание, удаление, просмотр активных процессов).
- Вывод информации о системе.
- Очистка консоли и базовые утилиты.

SamITOS является простым эмулятором ОС, который можно расширять, добавляя новые команды и функциональность.
